## 一、沙拉查词停更深度调查

### 1.1 停更时间线

| 时间 | 事件 |
|------|------|
| 2025年3月 | Chrome 开始禁用 Manifest V2 扩展 |
| 2025年5月 | Issue #2292 用户询问 MV3 迁移计划 |
| 2025年7月 | Chrome 138 完全无法使用 (#2296) |
| 2025年8月 | 大量用户反馈"扩展不再受支持" (#2298) |
| 2025年9月 | 有用户提议社区分叉开发 MV3 版本 (#2301) |
| 2025年10月 | 用户分享 ungoogled-chrome 变通方案 (#2304) |
| 2025年12月 | 彩云小译等词典源也相继失效 (#2311) |

### 1.2 停更原因分析

**核心原因: Manifest V2 → V3 迁移未完成**

从 GitHub Issues 分析:
1. **开发者 crimx 未公开说明停更原因** - 搜索全网未找到官方声明
2. **MV3 迁移工作量巨大** - 沙拉查词依赖大量 background scripts，MV3 的 service worker 模式需要重构
3. **个人项目维护难以为继** - 作为开源项目，维护投入产出比低
4. **词典 API 依赖不稳定** - 有道、彩云等第三方 API 频繁失效

**用户社区讨论关键信息:**
- Issue #2301: "不维护了么？chrome 高版本只能用 V3 了，有大佬一起改版成开源 v3 么"
- Issue #2286: "此扩展程序未遵循 Chrome 扩展程序的最佳实践，因此已无法再使用"
- 部分用户选择使用 ungoogled-chrome 等替代浏览器继续使用

### 1.3 用户功能需求遗留

从 GitHub Issues 提取的未满足需求:

| Issue # | 需求描述 | 与 LingoRecall 相关性 |
|---------|----------|----------------------|
| #2309 | 翻译后自动加入生词本 | ⭐⭐⭐ 高度相关 |
| #2307 | 保存单词所在句子/段落作为语境 | ⭐⭐⭐⭐ 核心功能契合 |
| #2287 | 支持 ChatGPT/DeepSeek API | ⭐⭐⭐ AI 集成方向 |

| #705 | 增加 DeepL 翻译 | ⭐⭐⭐ 多 AI 源需求 |

## 二、用户迁移去向分析

### 2.1 主要迁移目的地

| 竞品 | 用户规模 | 迁移原因 | 用户评价 |
|------|----------|----------|----------|
| **沉浸式翻译** | 300万+ | Chrome 官方推荐、MV3 兼容 | ⭐4.31 (1707评) |
| **划词翻译** | 30万+ | 轻量级、开源、MV3 兼容 | ⭐4.57 (1779评) |
| **Relingo** | 4万+ | 自动提取生词、视频双语字幕 | ⭐4.60 (943评) |

### 2.2 竞品功能缺口分析

**沉浸式翻译 (Immersive Translate)**
- ✅ 双语对照翻译
- ✅ PDF/视频翻译
- ✅ 多引擎支持 (20+)
- ❌ **无生词本功能**
- ❌ **无跳回原文功能**
- ❌ **无间隔复习系统**

**Relingo**
- ✅ 自动提取生词
- ✅ Netflix/YouTube 双语字幕
- ❌ **语境保存不完整**
- ❌ **无跳回原文高亮**
- ❌ 付费功能较多

**划词翻译**
- ✅ 开源、轻量
- ✅ 多引擎支持
- ❌ **无生词本**
- ❌ **无复习功能**

## 五、差异化机会与策略

### 5.1 LingoRecall 独特价值主张

**核心差异化: "语境记忆 + 跳回原文"**

```
竞品定位:    翻译工具 → 生词本 → 导出复习
LingoRecall: 翻译工具 → 语境锚定 → 原文跳回 → 智能复习 → 记忆强化
```

**独创功能:**
1. **XPath 原文定位** - 保存单词在网页中的精确位置
2. **一键跳回高亮** - 打开原网页并滚动到单词位置高亮显示
3. **语境完整存储** - 保存单词前后文、所在段落
4. **本地优先架构** - IndexedDB + chrome.storage.local

### 5.2 市场定位策略

**目标用户画像:**
- 英语学习者（雅思/托福/考研）
- 科研工作者（阅读外文文献）
- 程序员（阅读技术文档）
- 海外留学生/新移民

**竞争策略:**
1. **避开沉浸式翻译** - 不做全文翻译，专注"词汇语境记忆"
2. **吸纳沙拉查词用户** - 提供数据迁移工具
3. **差异化功能领先** - 跳回原文是杀手锏
4. **隐私安全优势** - 本地优先，可选云同步

### 5.3 风险评估

| 风险类型 | 描述 | 缓解措施 |
|----------|------|----------|
| **技术风险** | XPath 跳回在动态网页可能失效 | 备用定位策略(文本匹配) |
| **竞争风险** | 沉浸式翻译可能添加类似功能 | 快速迭代，建立用户粘性 |
| **用户获取** | 市场教育成本 | 清晰的产品 messaging |
| **维护风险** | 第三方 API 失效 | 多 AI 源 + 本地 Ollama |
